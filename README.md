## About

Blog developed to understand and demonstrate the Flexbox layout concepts of CSS.

The idea of this project it was to go deep on CSS concepts as:

## Flex Container:
- Display;
- Flex-direction;
- flex-wrap;
- flex-flow;
- justify-content;
- align-items;
- align-content;

## Flex Item:
- Flex-grow;
- Flex-basis;
- Flex-shrink;
- Flex;
- Order;
- Align-self
